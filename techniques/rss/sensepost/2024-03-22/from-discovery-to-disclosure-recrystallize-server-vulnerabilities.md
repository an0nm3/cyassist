TL&amp;DR  &#8211; While on an assessment, I found an instance of ReCrystallize Server. It had many problems, some of which had to do with insufficient hardening on the client&#8217;s side while others were new vulnerabilities I found that when chained together, achieve Remote Code Execution (RCE). These vulnerabilities were disclosed to ReCrystallize Software and MITRE. 
 Besides the disclosed vulnerabilities, some &#8220;features&#8221; were also used for malicious purposes. The replication and validation of the findings were done on my own test environment.
