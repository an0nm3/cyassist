If you have been doing internal assessments on Active Directory infrastructure you may have heard the following words: &#8220;Null session&#8221;, &#8220;Guest session&#8221; and &#8220;Anonymous session&#8221;. These words describe techniques that can be used on Windows servers to connect to resources and obtain information about a computer or Active Directory objects such as users or SMB shares you have access on. Even if these techniques are well known I realised that people, including myself, didn&#8217;t understand them completely. This got me wanting to better understand it, hence this blogpost :)!
