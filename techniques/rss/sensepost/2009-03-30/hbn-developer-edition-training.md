 Information about the course: 
 
 HBN &#8211; Developer Edition 
 
 &#8216;Hacking By Numbers &#8211; Developer Edition&#8217; is a course aimed at arming web application developers with knowledge of web application attack techniques currently being used in the &#8216;wild&#8217; and how to combat them. Derived from our internationally acclaimed &#8216;Hacking By Numbers&#8217; security training, this course focuses heavily on two questions: &#8220;What am I up against?&#8221; and &#8220;How can I protect my applications from attack?&#8221; During the course sample applications will be dissected to discover security related bugs hidden within the code. The class will then consider prevention, detection &amp; cure.
