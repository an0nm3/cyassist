Context 
 In the chilly month of December 2023, my colleagues Jason ( @BreakerOfSigns ), Szymon ( @TH3_GOAT_FARM3R ), and myself ( @felmoltor ) were on a red team. This one was tough, but we had fun. We had to be a bit more creative than I am used to and two interesting things were done that are worth sharing:&nbsp; 
 
 Szymon and Jason physically broke into the client’s facilities. At a branch office, they left an implant using Rogan’s  “Slimjim” device  and it proved to be a solid and production ready project.  Try it out and    let us know    how it goes. &nbsp; 
 I developed a tool that we named Mail-in-the-Middle (Maitm for short). You can find it in our Github repository here:  https://github.com/sensepost/mail-in-the-middle .&nbsp; 
 
 I cannot speak in first person about the physical breakin beyond praising the excellent social engineering skills of both of my colleagues, but I  can  talk about Mail-in-the-Middle.&nbsp;
