History 
 In December 2015  Silent Break Security  wrote about &#8220; Malicious Outlook Rules &#8221; and using these to get a remote shell. This was great, we could now use those credentials found through brute-forcing OWA instances or a phishing page. The only issue I had with this was the fact that you needed to setup a local instance of the mailbox, which at times could be time consuming and also felt like overkill.
