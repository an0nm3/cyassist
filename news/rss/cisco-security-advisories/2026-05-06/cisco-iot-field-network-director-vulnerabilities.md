Multiple vulnerabilities in the web-based management interface of Cisco IoT Field Network Director Software could allow an authenticated, remote attacker to access files, execute commands, and cause denial of service (DoS) conditions on managed routers. 
 For more information about these vulnerabilities, see the&nbsp; Details  section of this advisory. 
 Cisco has released software updates that address these vulnerabilities. There are no workarounds that address these vulnerabilities. 
 This advisory is available at the following link:  https://sec.cloudapps.cisco.com/security/center/content/CiscoSecurityAdvisory/cisco-sa-iot-fnd-dos-n8N26Q4u
