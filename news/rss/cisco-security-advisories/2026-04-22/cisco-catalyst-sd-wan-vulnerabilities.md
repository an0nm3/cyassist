Multiple vulnerabilities in Cisco Catalyst SD-WAN Manager, formerly SD-WAN vManage, could allow an attacker to access an affected system, elevate privileges to&nbsp; root,  gain access to sensitive information, and overwrite arbitrary files.&nbsp; 
 For more information about these vulnerabilities, see the  Details  section of this advisory. 
 Cisco has released software updates that address these vulnerabilities. There are no workarounds that address these vulnerabilities. 
 Cisco strongly recommends that customers upgrade to the fixed software indicated in this advisory. 
 This advisory is available at the following link:  https://sec.cloudapps.cisco.com/security/center/content/CiscoSecurityAdvisory/cisco-sa-sdwan-authbp-qwCX8D4v
