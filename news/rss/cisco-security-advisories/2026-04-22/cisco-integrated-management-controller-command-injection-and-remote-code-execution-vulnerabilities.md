Multiple vulnerabilities in the web-based management interface of Cisco Integrated Management Controller (IMC) could allow an authenticated, remote attacker to execute arbitrary code or commands on the underlying operating system of an affected system and elevate privileges to&nbsp; root . 
 For more information about these vulnerabilities, see the&nbsp; Details  section of this advisory. 
 Cisco has released software updates that address these vulnerabilities. There are no workarounds that address these vulnerabilities. 
 This advisory is available at the following link:  https://sec.cloudapps.cisco.com/security/center/content/CiscoSecurityAdvisory/cisco-sa-cimc-cmd-inj-3hKN3bVt
